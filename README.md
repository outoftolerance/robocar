# Robocar
This is my robocar project code. It sucks and I am not a great C++ programmer. Have fun!

### Installation

Robocar requires [OpenCV 3](https://opecv.org)

Clone
```sh
$ git clone http://github.com/outoftolerance/robocar
```

Build
```sh
$ cd robocar
$ cmake .
$ make
```

Run
```sh
$ ./robocar
```
